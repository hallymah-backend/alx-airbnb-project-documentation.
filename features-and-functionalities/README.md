# features-and-functionalities

This directory contains the backend features documentation and a diagram for the Airbnb Clone project.

Files:
- FEATURES_AND_FUNCTIONALITIES.md — detailed features list
- airbnb_backend_features.png — visual diagram of components

## How to add to your repository `alx-airbnb-project-documentation`

From your local clone of the repository:

```bash
# create directory (if not present)
mkdir -p features-and-functionalities
# copy files into your repo folder, then from repo root:
git add features-and-functionalities/FEATURES_AND_FUNCTIONALITIES.md features-and-functionalities/airbnb_backend_features.png features-and-functionalities/README.md
git commit -m "Add backend features documentation and diagram for Airbnb Clone"
git push origin main
```

Adjust branch name (`main`) as needed.
